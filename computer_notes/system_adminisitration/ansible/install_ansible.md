# 在Ubuntu上安装Ansible

什么是Ansible    
-------------------------------------
  ansible是远程自动控制服务器。     

Ansible需要的依赖    
------------------------------------
  ansible是基于python3开发的。所以按检查系统是否有安装一般都会自带python    
  `which python3 //会显示安装python3的位置`  
  Ansible使用pythhon的pip来安装的。    
  `sudo apt install python3-pip`

安装Ansible     
------------------------------
  `pip install --user ansible`
