# HTML文本基础
  大部分結構化文本由標題和段落組成的。

標題和段落     
-----------------------------------------------------------
在HTML中，每個段落是通過`<p>`元素標簽來定義的。    
`<p>我是一個段落，千真萬確</p>`    

每個標題(Heading)都必須被包裹在一個標題元素中：     
`<h1>我是文章標題</h1>`     
標題元素有六種 `h1, h2, h3, h4, h5, h6`。每個元素代表文檔中不同級別的内容。
